## Backtester Requirements

The Python backtester must replicate:

1. **ORB Detection**
   - Track session high/low during ORB period
   - Handle retroactive ORB calculation

2. **Breakout Logic**
   - Close above/below ORB ± (ATR × threshold)
   - Body strength ratio check
   - ORB range min/max filters

3. **Confluence Scoring**
   - SSL confirmation (price vs baseline + acceleration)
   - WAE confirmation (trend > explosion line + acceleration)
   - QQE confirmation (RSI conditions + momentum direction)
   - Volume scoring

4. **Stop Loss Calculation**
   - ATR-based stops
   - Swing high/low stops (RTH only)
   - VWAP-based stops
   - Hybrid selection logic
   - Auto-select best R:R

5. **Exit Management**
   - Break-even move at target R:R
   - EMA cross exit (with confirmation bars, body %)
   - Trailing stop activation and tracking
   - Aggressive trailing near EMA

6. **Adaptive Volatility**
   - Vol factor calculation (session × HTF)
   - Vol state classification
   - R:R adjustment per regime
   - Min acceptable R:R gate
